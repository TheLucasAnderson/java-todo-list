## Description
This is a To-Do List application project developed in Java using the Spring framework and managed by Maven. The application allows users to create, view, update, and delete tasks.

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Project Setup

### Prerequisites
- Java 17 or higher
- Maven 3.6.0 or higher
